## About
Learn what ensemble methods are, how to fit and evaluate bagged decision trees and random forests, and how to compare their performance, benefits, limitations, and underlying rationale.


## Learning Objectives

*You will be able to:*

- Define ensemble models and when to use them.
- Describe how and why decision trees can be improved using bagging and random forests.
- Fit and evaluate bagged decision trees and random forests.
- Understand how to calculate out-of-bag error.
- Explain how to extract the most important predictors in a random forest model.

## Prerequisites
- Construct individual decision trees.
- Describe how CART models are built.
- Differentiate between classification and regression problems.
- Understand the bias-variance tradeoff.
